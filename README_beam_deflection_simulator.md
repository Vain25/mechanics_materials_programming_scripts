Beam Deflection Simulator
This Python tool calculates and visualizes the vertical deflection of a simply supported bridge beam under a uniform load.
Features
User Inputs: Allows custom bridge name, length, material properties (Young's Modulus), and loading.
Physics-Based: Uses the elastic curve equation for accurate deflection modeling.
Visualization: Generates a publication-quality graph and saves it as a PNG.
How to Run
Install dependencies: pip install numpy matplotlib scipyRun the script: python beam_plot.py
